Message Popup

Custom success, warning, info, error messages popup.

Usage

How to use messages popup

In your model o wizard function return the message as follow:

return self.env['message.popup'].popup(message_type="info", message="Your message here")

message_type is optional and by default is "success". Possible values are success, warning, info, error.
